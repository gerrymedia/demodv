An interactive web application allowing users to vote thumbs up/down on multiple images. 

Requirements:
Each vote should prompt the next image and update percentages. 
Each user should only be able to vote once per image.

Developed using:
* Normalized MySQL
* PHP Object Oriented approach
* MVC design
* Cross browser compatible HTML/CSS
* PHP5 - autoloading, namespace
* MYSQLi Object Oriented approach
* Frontend: Tactile user interface
